Introduction

This project focuses on integrating, cleaning, and analyzing data from Indian Airlines, to analyze the price variations, Airline prefereces by customers and route-based analysis. By merging multiple CSV files and conducting SQL-based analysis, the project aims to provide actionable insights for pricing strategies, service enhancements, and operational decisions.

Objective

  • Data(4.31L+ rows) containing booking details data into a single DataFrame.
  • Perform data cleaning to handle missing values, duplicates, and standardize formats.
  • Analyze pricing patterns among various Airlines and routes, focusing on booking pattern and behaviour.
  • Identify customer prefereces based on Airline, fare, Journey-duration, and price variations due to Booking status, no.of stops etc.
  • Generate insights to inform pricing strategies, service improvements, and resource allocation.

Methodology

  1. Data Collection

    The CSV files containing booking details data is available in Kaggle.

  2. Data Integration

    The os and Pandas library in Python is utilized to merge all CSV files into a single DataFrame using the pd.concat() function, ensuring that no data is lost during the merging process.

  3. Data Cleaning

    Null Value Handling: Missing values in the dataset are identified and addressed using the fillna() method. Object-type columns such as station names and IDs are filled with "not specified", while float-type columns such as geographical coordinates are filled with a placeholder value (-999).

    Duplicate Detection: Duplicate rows in the dataset are identified using the duplicated() method.

  4. Data Transformation

    Date and time columns are converted to datetime format using the pd.to_datetime() function for consistency and ease of analysis.

  5. Data Export

    The cleaned and integrated dataset is saved as a new CSV file named "merged_data.csv" using the to_csv() method.

  6. Data Analysis in SQL

    Data Import

    The dataset is imported into a PostgreSQL database table named "cyclist" using the COPY command.

    Data Transformation

    Created a new table in Power BI using the existing data with the column name Route by combing the source and destination columns. Calculated Average fare, min fare, max fare etc.. for each route. And created a new table for Airlines and calculated avg fare, min fare, max fare etc for each airlines.

    Data Cleaning

    Null values are checked and consistent values are ensured for categorical variables such as Fare, Source, Destination CHecked for data types.

  7. Analysis

    General Overview:

    Breakdown of bookings by Airline, Route and class prefereces.

    Average ticket price for each airline and for each route.

    Price comparison based on booking class.

    Total revenue generated by each Airline per each route.

    Total Number of Bookings:

    Analysis of bookings trends, timing preferenfce of travel of customers, pricing comparison based on how price vary based on ticket booking days before travel and cancellation reason as a pie chart .

    Average Fare per class and total bookings per Airline:

    Analysis of average, max, min fare of each airline, also predicted the April month data based on the previous three month data.

Result

  1. Data cleaning operations addressed missing values and removed duplicate rows, ensuring the quality and integrity of the dataset.
  2. The cleaned dataset is ready for exploratory data analysis, modeling, and generating insights into Pricing patterns and booking trends.
  3. General Overview:
    • Economic class are the most prefered choice by customers, but business class generated more revenue than economy class despite less bookings due to huge price difference between each class.
    • Most customers prefer day-time travel, advance-bookings due to lower price when compared to last-minute booking.
  4. Total Number of Bookings:
    • Vistara Airline is the most prefered Airline for domestic customers and Delhi-Mumbai is the busiest route by Bookings.
    • Casual riders show a preference for direct flights, with short-duration flights, with the age-group 28-45 are the most travellig group.
  5. Average Bookings:
    • Though average bookings remain the same considering the booking day, the price increases with respect to the journey day, advance booking has relatively lower price comapred to last-minute nooking.
