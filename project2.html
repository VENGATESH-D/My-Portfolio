Airfare Analysis in India

Data Analytics Capstone Project

  1. Home
  2. Airfare Analysis in India

Objective

This project focuses on integrating, cleaning, and analyzing data from Indian Airlines, to analyze the price variations, Airline prefereces by customers and route-based analysis. By merging multiple CSV files and conducting SQL-based analysis, the project aims to provide actionable insights for pricing strategies, service enhancements, and operational decisions.

Methodology

  1. Data Transformation:

    The project begins with data acquisition and transformation. Cleaned and Merged data is imported into Power BI and transformed to ensure consistency and usability.

    Column values are standardized by replacing data_type such as "Date_of_Journey" from text type to date type.

    Additional columns are created to extract meaningful information, including month names, Route, Age, Payment_Mode, Gender.

  2. Data Modeling and Measures:

    Key performance indicators (KPIs) are defined, such as Average_fare and Total_Revenue.

    Measures are created using Data Analysis Expressions (DAX) to calculate metrics such as Total_Revenue per Airlines and Fare_per_Route.

  3. Visualization Creation:

    Various visualizations are developed to provide insights into Pricing & Booking patterns.

    Created table to calculate route and total revenue per route, created a tile slicer for month wise data.

    Line charts visualize trends in price variations as per booking day with respect to journey day.

  4. Dashboard Development:

    Four distinct dashboards are created: Summary, Class wise analysis, Booking trends & Booking Analysis.

    Each dashboard offers specific insights and facilitates easy navigation between different sections, enhancing user experience and understanding.

  5. Interactivity and Navigation:

    Navigation buttons are integrated to enable seamless navigation between dashboards, allowing users to explore various analyses and insights effortlessly.
